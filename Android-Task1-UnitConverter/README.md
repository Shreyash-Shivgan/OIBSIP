# Task 1 - Unit Converter

A professional Unit Converter application for Android built with Java and XML.

## Features
- **Dynamic Category Selection**: Length, Weight, and Temperature.
- **Auto-Updating Dropdowns**: When the category changes, the source/target units update automatically.
- **Input Validation**: Prevents empty input, invalid character formats, and shows immediate TextInputLayout error feedback.
- **Accurate Conversions**: Precise conversions utilizing well-defined constants and base unit normalization (meters, grams, Celsius).
- **Modern UI**: Styled card views with custom indigo-teal accent tones.

## Tech Stack
- **Languages**: Java, XML
- **Minimum SDK**: 24
- **Target SDK**: 34
- **Layout**: ConstraintLayout, LinearLayout, CardView
- **Theme**: Material Design 3

## Installation
1. Download or clone this folder.
2. Open Android Studio and select **Open**.
3. Select the `Android-Task1-UnitConverter` directory.
4. Let Gradle sync completely.
5. Click the **Run** button to deploy on an emulator or active Android device.
