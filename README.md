## Completed Tasks

### 📦 [Task 1 - Unit Converter](file:///C:/Users/shrey/.gemini/antigravity-ide/scratch/OIBSIP/Android-Task1-UnitConverter)
- **Description**: A smart converter for Length, Weight, and Temperature with dynamic dropdown validation and professional Material Design 3 cards.
- **Tech Stack**: Java, XML, ConstraintLayout, Material Components.

### 📦 [Task 2 - ToDo App With Login](file:///C:/Users/shrey/.gemini/antigravity-ide/scratch/OIBSIP/Android-Task2-ToDoAppWithLogin)
- **Description**: A secure Task Manager supporting user registration, SHA-256 hashed login authentication, SharedPreferences sessions, and user-specific SQLite CRUD operations.
- **Tech Stack**: Java, XML, SQLite, SharedPreferences, SHA-256 Hashing, RecyclerView.

### 📦 [Task 3 - Calculator](file:///C:/Users/shrey/.gemini/antigravity-ide/scratch/OIBSIP/Android-Task3-Calculator)
- **Description**: A modern, dark-themed expression calculator with division-by-zero protection, decimal checks, backspace, and consecutive operations parser.
- **Tech Stack**: Java, XML, GridLayout.

### 📦 [Task 4 - Quiz Application](file:///C:/Users/shrey/.gemini/antigravity-ide/scratch/OIBSIP/Android-Task4-QuizApplication)
- **Description**: A 10-question general knowledge multiple-choice quiz with option shuffling, immediate correct (green) / incorrect (red) feedback, and a summary scoreboard.
- **Tech Stack**: Java, XML, CardViews, Color Animators.

### 📦 [Task 5 - Stopwatch](file:///C:/Users/shrey/.gemini/antigravity-ide/scratch/OIBSIP/Android-Task5-Stopwatch)
- **Description**: An advanced, lifecycle-safe stopwatch using standard Java Handlers, featuring start, pause, reset, and lap history tracking.
- **Tech Stack**: Java, XML, RecyclerView, SharedPreferences.

---

## Tech Stack Overview
- **Language**: Java 17
- **UI Markup**: Android XML Layouts
- **Layout Manager**: ConstraintLayout, GridLayout
- **Material Theme**: Material Design 3 (M3)
- **Minimum SDK**: 24 (Android 7.0)
- **Target SDK**: 34 (Android 14)
- **Database**: SQLite (local database)
- **Session Management**: SharedPreferences
- **Build System**: Gradle 8.2.2+
